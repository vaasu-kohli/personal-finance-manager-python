# Personal Finance Manager (Python)

A simple command-line personal finance manager built using Python.  
This project allows users to record income and expenses, analyze spending by category, and visualize expense distribution using a pie chart.

## Features
- Add income
- Add expenses with categories
- View all transactions
- View summary (income, expenses, balance)
- Category-wise expense analysis
- Expense pie chart visualization
- JSON-based local data storage

## Technologies Used
- Python
- JSON
- Matplotlib

## How to Run
python3 finance_manager.py

## Notes
This is a beginner-level Python project focused on learning file handling, functions, and basic data visualization.
